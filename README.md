
# DeskFit - exercises for sitting worker

DeskFit is a fitness web application specifically for sitting workers to promote physical health by providing easy access equipment-free exercises during their workday. 

**Key Features :**  
* Expert-Guided Exercise Videos – Users receive guidance through videos created by fitness experts, focusing on exercises suitable for desk-bound workers.

* Step-by-Step Instructions – Clear instructions accompany each exercise to ensure proper form and effectiveness.

* Focus Area & Breathing Tips – Additional guidance on focus areas for the exercises and breathing techniques to maximize health benefits.

**hosted link** : https://desk-fit.vercel.app/


## Tech Stack

**Frontend :** React, TailwindCSS

**Backend :** Node, Express, MongoDB
